FFXIV Custom Plugin Repository

This repository is for testing! You should just install the released plugin you want to use from the official list in-game.

The plugins in this repo are in their pre-testing state. They are subject to rapid changes and all kinds of errors, including game crashes. If you are here to help me test these plugins before release into Dalamud testing, continue reading.

To access the plugins in this repo, do the following:

  1. Open the Dalamud settings menu in game.
  2. Choose the Experimental tab.
  3. In the Custom Plugin Repositories section, add https://markjsosnowski.github.io/FFXIV/plugin_repo.json to the list.
  4. Save. You may have to restart your game to see the plugins added to the list.

If you encounter any issues with these testing plugins, please report them so I can get it fixed before release.

Once/If the plugin is accepted into the offical Dalamud repo, you should uninstall the custom version and add the release version.
